### 🎯 **What This Project Does**

This AI-powered system uses **Conditional Random Fields (CRF)** to solve one of the most expensive problems in logistics: **Messy Address Data**. 

Whether it's the lack of spaces (`123mainst`), typos (`mian rd`), or complex Indian local markers (`Plot 34, Opp Plaza`), our model tokenizes and classifies every character into meaningful shipping components.

**The Complete Pipeline:**
- 📡 **Data Synthesis** → Custom "Messy Data" generator (100k+ rows)
- 🔄 **Feature Engineering** → N-gram character windows & word anchors
- 🧠 **Model Training** → Linear-Chain CRF for sequence labeling
- 📊 **UI Engine** → Glassmorphism Streamlit Dashboard
- 📦 **Bulk API** → CSV batch processor for enterprise scale
